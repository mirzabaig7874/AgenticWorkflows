Overview

Every 5 hours, this flow checks a Google Sheet that logs junk emails from the Inbox Cleaner workflow. If there's data, it formats the entries into a clean AI-generated HTML table, emails you a report, then wipes the sheet clean — ready for the next batch.

How It Works

⏱️ Scheduled Trigger Runs automatically every 5 hours
↓
🔢 Count Sheet Rows Checks how many junk entries are logged
🔗 Sheet URL Constant — passed into router
↓
🔀 Router: Is Sheet Empty? Checks row count and decides which path to take
↙
Rows = 0
🛑 Stop Sheet is empty — flow exits
↘
Rows > 0
📋 Read Sheet Data Pulls Subject, Sender, Date columns
↓
🔧 Structure Rows 📧 Subject · 👤 Sender · 📅 Date per row
↓
🔗 Join All Rows Combines into a single text block
↓
🤖 AI → HTML Table Formats text into a clean HTML table
↓
📧 Send Email Report "🗑️ Junk Mail Report — Last 5 Hours"
↓
🧹 Clear Sheet Wipes rows — ready for next batch

Step-by-Step Breakdown

1–2
Count Rows + Sheet URL
Reads the row count from the junk email Google Sheet and stores the sheet URL as a reusable constant — both feed into the router.
3
Conditional Router
If row count is 0 → flow stops (nothing to report). If rows exist → continues to read and process the data.
4–6
Read, Structure & Join
Reads the sheet (Subject, Sender, Date), formats each row into a readable string, then joins all rows into a single text block.
7
AI → HTML Table
AI converts the plain-text row block into a polished HTML table with Sender, Subject, and Date columns — ready to embed in an email.
8
Email the Report
Sends the formatted HTML table to mirza.asad.baig@gmail.com with subject "🗑️ Junk Mail Report — Last 5 Hours".
9
Clear the Sheet
Wipes all rows from the Google Sheet after sending — resetting it for the next 5-hour cycle.
🔁 Works in tandem with Inbox Cleaner — that flow logs junk emails to the sheet; this flow reports and resets it every 5 hours.
